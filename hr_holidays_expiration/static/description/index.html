Holidays expiration

Limit the leave backlog.

Your employees' leaves are subject to expiration. Employees should usually not be able to claim leave rights from years ago. We created the expiration concept on leaves, so leaves will expire after a set period.

When creating leave types you can now can decide to make the leaves valid only for a certain period by setting the validation period. You can specify that the persons responsible for managing leaves are informed automatically by email.

The leave request will expire according to settings defined in the selected Leave Type.
